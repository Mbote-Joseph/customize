## Get Started
Install `node-modules` with:
```bash
yarn
```

Get up and running:
```bash
yarn start
```

Building a hostable package:
```bash
yarn build
```

Easily deploy with [zeit.co](https://zeit.co)'s Now.
```bash
cd build
now -n yourChoiceOfSubdomain
```
